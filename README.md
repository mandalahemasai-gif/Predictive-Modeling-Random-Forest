# Predictive-Modeling-Random-Forest
# Predictive Modeling Using Machine Learning 🤖

## Overview
This project demonstrates supervised machine learning by building a predictive model to classify outcomes based on a given dataset. It covers the full pipeline from data splitting to model training and performance evaluation.

## Key Features
* **Algorithm Used:** `Random Forest Classifier` (an ensemble learning method).
* **Training & Testing:** Implemented a 70/30 train-test split using `scikit-learn` to ensure unbiased evaluation of the model's accuracy.
* **Performance Visualization:** Evaluated the model's predictive power using a **Confusion Matrix** (to analyze false positives/negatives) and an **ROC Curve** (to measure the area under the curve/AUC).
